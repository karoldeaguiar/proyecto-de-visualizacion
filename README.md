# Proyecto de Visualización de Datos Clínicos

## Descripción
Este proyecto integra datos procedentes de ClinicalTrials.gov y PubMed
para analizar ensayos clínicos y publicaciones científicas relacionadas
con cáncer de pulmón no microcítico y tratamientos quirúrgicos e inmunoterapia.

## Objetivo
Construir un dataset unificado y una visualización interactiva que permita
explorar tendencias clínicas, geográficas y documentales.

## Estructura del repositorio
- notebooks/: notebooks de descarga, limpieza y análisis
- dashboard/: material visual del dashboard
- docs/: documentación adicional del proyecto
